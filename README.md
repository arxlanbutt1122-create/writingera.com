# WritingEra - Professional Writing Services Platform

A modern, responsive web application for WritingEra's academic and business writing services.

## About

WritingEra provides expert writing services including essays, research papers, thesis writing, dissertations, business content, and more. This platform offers a seamless experience for clients to explore services, place orders, and get professional writing assistance.

## Technologies

This project is built with:

- **React** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI components
- **React Router** - Navigation

## Features

- 🎨 Beautiful, responsive design
- 📱 Mobile-first approach
- 🌐 Multi-currency support (PKR/USD)
- 🔍 SEO optimized
